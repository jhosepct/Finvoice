Ejercicio 3

¿Qué significa HTML, y cuál es su estructura básica?

Respuesta

HTML significa HyperText Markup Language, es un lenguaje de marcado que se utiliza para el desarrollo de páginas web. Su estructura básica es la siguiente: 

        <!DOCTYPE html>
        <html lang="en">
          <head>
            <meta charset="UTF-8" />
            <meta http-equiv="X-UA-Compatible" content="IE=edge" />
            <meta name="viewport" content="width=device-width, initial-scale=1.0" />
            <title>Título de la página</title>
          </head>
          <body>
            <h1>Título</h1>
            <p>Contenido</p>
          </body>
        </html>
